# Assets Directory

This directory is for storing images, icons, and other media files for the portfolio website.

## Directory Structure

```
assets/
├── images/
│   ├── profile.jpg          # Your profile picture
│   └── projects/            # Project screenshots
│       ├── project1.jpg
│       ├── project2.jpg
│       └── ...
└── icons/                   # Custom icons (if needed)
```

## Current Setup

- **Icons**: Currently using Font Awesome CDN (no local files needed)
- **Images**: Using placeholder images from via.placeholder.com
- **Replace placeholders**: Update image sources in `index.html` with your actual images

## Image Recommendations

- **Profile Image**: 400x400px, square format, optimized for web
- **Project Images**: 600x400px, landscape format, optimized for web
- **Format**: Use JPG for photos, PNG for graphics with transparency
- **Optimization**: Compress images before uploading to improve load times
